# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - Release candidate

- Added the `connector-route-replay` CLI for replaying local connector-routing fixtures.
- Added fixture verification against expected selected routes and approval gates.
- Added deterministic scoring for read-only, dry-run, write, blocked, and ambiguous connector paths.
- Added fixture-backed tests, smoke checks, CI verification, and npm pack dry-run coverage.
